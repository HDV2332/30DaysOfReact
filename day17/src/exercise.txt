    3.What is the use of the following Components(Route, NavLink, Switch, Redirect, Prompt)
Route: allows to navigate between components.
NavLink: allow us to navigate each component. It takes a to required props.
Switch: allows only on component to be rendered.
Redirect: help us to redirect a route to a certain path based some condition.
Prompt: prompt the user before navigating away from a page.
